# Social media API
## About
Social media API with authentication and authorization of users,
managing posts and writing comments. Authentication is implemented
using JWT Bearer tokens.
## Used technologies
- Spring Boot 3.0.0
- Spring Security 3.0.0
- Oauth2 Resource Server
- JWT
- Spring Data JPA
- OpenAPI
- Docker
## Docker
    docker compose up
